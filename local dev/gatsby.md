## Steps
1. Install Gatsby CLI globally https://www.gatsbyjs.com/docs/tutorial/getting-started/part-0/

    Node.js (v18 or newer, but less than v21)
    ```
    npm install -g gatsby-cli
    ```

2. Create Gatsby Project: 
    ```
    gatsby new gatsby-site-name
    ```

3. Run Gatsby in development from project dir:
    ```
    gatsby develop
    ```

4. Install Gatsby Source for Drupal https://www.gatsbyjs.com/plugins/gatsby-source-drupal/
    ```
    npm install gatsby-source-drupal
    ```

5. Enable modules in drupal
    - JSON:API
    - JSON:API Extras
    - Gatsby
    - Gatsby JSON:API Extras
